    • APIとは

    リクエスト&レスポンスにより、ユーザの変わりにプログラムに機能やサービスを利用させる
    開発コスト削減やAPI連携によるシステム・サービスの質の向上などが期待できる

    APIの利用例

    • LINE：Messaging API　LINEで自動返信ボットなどを作ることができる
    • Twitter：ツイート内容を分析することでユーザーのトレンド分析に活用
    • Youtube：いいね数などの動画情報の取得や年齢層などの動画の統計情報などを取得
